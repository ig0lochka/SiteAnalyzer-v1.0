# SiteAnalyzer Web v1.02

### Основные возможности
- Семантический анализ заголовков и контента (ru-sentence-bert с fallback на TF-IDF)
- Поддержка доменов .рф и кириллических URL
- Измерение времени загрузки страницы с оценкой скорости
- Авторизация пользователей (выпадающий список + поле пароля с поддержкой Ctrl+V)
- Генерация структурированных отчётов в Excel
- Автоматическое логирование ошибок в error_log.txt
- Полностью автономный режим — не требует установки Python

### Требования
- Windows 10/11
- 8 ГБ ОЗУ
- Intel Core i3 или выше

### Установка
1. Скачать `SiteAnalyzer Web v1.02.exe`
2. Запустить двойным кликом
3. Войти: логин `admin` | пароль `Admin2026!`

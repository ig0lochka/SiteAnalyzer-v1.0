SiteAnalyzer Web — автономное настольное приложение для проверки соответствия веб-сайтов учреждений дополнительного образования нормативным требованиям (Приказ Минпросвещения РФ № 659, ФГОС ДО, региональные стандарты).
Программа работает полностью оффлайн после первого запуска (модель анализа загружается один раз и сохраняется локально), не требует установки Python и дополнительных библиотек.
Установка и запуск

Скачайте файл SiteAnalyzer Web v1.06.exe из репозитория GitHub (раздел Releases):
https://github.com/ig0lochka/SiteAnalyzer-v1.0/releases/latest
Запустите файл двойным кликом.
Программа откроется сразу (окно входа).

Вход в систему

В выпадающем списке выберите пользователя (admin, ikt, metodist, sheptunova).
Введите пароль (можно скопировать из блокнота и вставить: Ctrl+V или правой кнопкой мыши → Вставить).
Нажмите кнопку ВОЙТИ.

Примеры паролей:

admin → Admin2026!
ikt → Ikt2026!
metodist → Metodist2026!
sheptunova → 12345

Как провести анализ сайта

В главном окне в поле Адрес сайта введите URL (например, dkymkk.ru или https://dkymkk.ru).
Нажмите кнопку ЗАПУСТИТЬ АНАЛИЗ.
Программа:
загрузит страницу (с таймаутом 25 секунд);
измерит время загрузки и оценит скорость («Отлично», «Хорошо», «Удовлетворительно», «Плохо»);
проверит наличие обязательных разделов («О нас», «Документы», «Контакты», «Программы», «Обратная связь», «Лицензия»);
рассчитает общий рейтинг соответствия;
сохранит результат в файл Excel в папке reports (имя файла с датой и временем).


Просмотр результатов

Результаты сразу отображаются в текстовом поле внизу окна.
Время загрузки, оценка скорости и рейтинг выводятся вверху.
Отчёт автоматически сохраняется в папку reports рядом с программой.
Нажмите кнопку Открыть отчёт — откроется папка с файлами Excel.

Логирование ошибок
Все ошибки и важные события записываются в файл error_log.txt в папке logs (рядом с программой).
Если анализ не прошёл — откройте этот файл и отправьте его разработчику.
Дополнительно

Программа не требует интернета после первого запуска (модель анализа кэшируется локально).
Работает на стандартных ПК учреждения (Windows 10/11, 8 ГБ ОЗУ).
Все данные остаются на вашем компьютере — конфиденциальность обеспечена.

# SiteAnalyzer Web v1.0

**Автоматическая проверка сайтов учреждений дополнительного образования Краснодарского края**

### Возможности
- Семантический анализ разделов (ru-sentence-bert)
- Поддержка доменов .рф
- Анализ времени загрузки
- Генерация отчётов в Excel
- Полностью автономен (без Python и интернета)

### Запуск
1. Скачать `SiteAnalyzer Web v1.0.exe` из Releases
2. Запустить двойным кликом
3. Логин: `admin` | Пароль: `Admin2026!`

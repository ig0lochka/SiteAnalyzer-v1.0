# SiteAnalyzer Web v1.05

### Основные возможности

## Умный поиск по смыслу (DL-модель ru-sentence-bert)
Раньше программа искала точные слова — если вместо «О нас» написано «Про центр» или «Сведения об учреждении», она могла не найти.
Теперь программа понимает смысл текста:
«Про центр», «О центре», «Информация об учреждении» → считает, что это всё равно «О нас»
«Нормативные документы», «Лицензии и устав» → считает «Документы»
Это сделано с помощью нейронной сети (модель ru-sentence-bert). Она превращает текст в математические числа (эмбеддинги), сравнивает их и говорит, насколько похожи фразы.

## Резервный поиск (fallback TF-IDF)
Если компьютер слабый или модель не загрузилась — программа автоматически переключается на более простой, но надёжный поиск по словам (TF-IDF + логистическая регрессия). Он быстрее и почти не требует памяти.

## Улучшенные отчёты
В Excel теперь пишется не только «Найден / Отсутствует», но и:
время загрузки сайта
оценка скорости («Отлично», «Хорошо» и т.д.)

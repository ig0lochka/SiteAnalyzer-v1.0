# Changelog SiteAnalyzer Web

Все заметные изменения в проекте документируются в этом файле.

## [1.0] - 2025-12-06

### Added
- Полноценный графический интерфейс на Tkinter
- Авторизация пользователей с выпадающим списком логинов
- Поддержка вставки пароля (Ctrl+V и правой кнопкой мыши)
- Анализ времени загрузки страницы с оценкой скорости
- Генерация структурированных отчётов в Excel
- Автоматическое логирование ошибок в error_log.txt
- Полная поддержка доменов .рф
- Резервный TF-IDF-классификатор (fallback при нехватке ОЗУ)

## [1.0.2] - 2026-01-03

### Added
- Полная интеграция ru-sentence-bert (ONNX) для семантической классификации
- Резервный TF-IDF + LogisticRegression классификатор (fallback при нехватке ОЗУ)
- Автоматическое переключение между DL и fallback
- Кэширование модели при запуске
